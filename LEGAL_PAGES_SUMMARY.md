## Design Features

### Layout
- **Two-column layout**: Sticky sidebar navigation + main content area
- **Responsive**: Max-width 1200px container with proper spacing
- **Clean header**: Logo on left, "Back to Home" link on right

### Sidebar Navigation
- **Active state**: Purple background (#f5f3ff) with purple text (#7C3AED)
- **Hover states**: Smooth transitions on inactive links
- **Sticky positioning**: Sidebar stays visible while scrolling

### Content Styling
- **Icon badges**: Gradient backgrounds with white icons
- **Typography hierarchy**: 
  - H1: 32px, weight 900
  - H2: 20px, weight 800
  - H3: 16px, weight 700
  - Body: 15px, line-height 1.8
- **Color scheme**:
  - Headings: #1a1a2e (dark)
  - Body text: #4b5563 (gray)
  - Meta text: #9999b0 (light gray)
  - Links: #7C3AED (purple)
- **Contact boxes**: Light gray background with border
- **Footer**: Centered copyright notice with top border

### Icons
